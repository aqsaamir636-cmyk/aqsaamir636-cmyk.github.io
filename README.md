## 📚 پروج��کٹس

### 1️⃣ **Aqsa AI Chatbot** 💬
- **مقصد**: اردو میں بات چیت کریں، سوالات پوچھیں، معلومات حاصل کریں
- **فیچرز**:
  - ✅ اردو/انگریزی میں بات چیت
  - ✅ وقت اور تاریخ کی معلومات
  - ✅ دعائیں اور احادیث
  - ✅ صارف کا نام یاد رکھتا ہے (localStorage میں)
  - ✅ وقت کے مطابق خوش آمدید پیغام
  - ✅ خوبصورت UI اور animations

**لائیو دیکھیں**: [Aqsa AI Bot](https://aqsaamir636-cmyk.github.io/My-aibot/index.html)

### 2️⃣ **خنداک بٹ - Joke Generator** 😂
- **مقصد**: Random jokes حاصل کریں اور ہنسیں!
- **فیچرز**:
  - ✅ تین قسمیں: عام، پروگرامنگ، Knock-Knock
  - ✅ External API (JokeAPI) سے jokes
  - ✅ پسندیدہ میں محفوظ کریں
  - ✅ شیئر کریں دوستوں کے ساتھ
  - ✅ شمار دیکھیں (کتنے jokes دیکھے)

**لائیو دیکھیں**: [خنداک بٹ](https://aqsaamir636-cmyk.github.io/My-aibot/joke-generator.html)

## 🛠️ استعمال کریں

### **Local میں چلانا:**

```bash
# اگر Python 3 ہے تو:
python -m http.server 8000

# یا Python 2:
python -m SimpleHTTPServer 8000

# پھر براؤزر میں کھولیں:
http://localhost:8000
```

### **npm سے:**

```bash
npm install
npm start
```
